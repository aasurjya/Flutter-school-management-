## 2. DELIVERY RULES

| Rule | Why |
|------|-----|
| Speak **10% slower** than feels natural | Demo nerves make people rush. Slow = confident. |
| **One thought per breath** | Cut sentences short. Use "/" pauses. |
| **Smile at start of each Act** | Listeners *hear* the smile. Energy resets. |
| **Drop pitch on key claims** | "40% reduction in dropout" — drop voice, don't raise. Drop = certain. |
| **No fillers** | "um", "like", "basically" — pause instead. Silence sounds smarter. |
| **Read numbers as stories** | Not "13,000% ROI" — say *"pays for itself in 3 days"*. |
| **Use 'you' and 'your school'** | "Your teachers save 20 hours a week" beats "Teachers save…". |
| **Never apologize** | No "this is still in beta", "we're working on it". |
| **End every Act with a benefit, not a feature** | Feature: PDF export. Benefit: ready to email parents in one click. |

## 3. FEATURES TO EMPHASIZE vs SKIP

### ✅ SHOW PROUDLY
- Multi-role login (15 accounts = real build, not vaporware)
- Role-based dashboards (visual variety = wow factor)
- Glassmorphism / Material 3 UI (looks premium on camera)
- Attendance marking (fast = impressive)
- Fee invoice PDF with ₹ (recently polished — flaunt it)
- Noah's AI risk flag (the *only* seeded AI demo data — make it count)
- Parent child-switcher (emotional hook)
- Multi-tenant + RLS (mention once, don't go deep)

### ⚠️ MENTION, DON'T DEEP-DIVE
- Offline sync (Isar models exist, sync logic still in progress — say *"works offline"*, move on)
- Analytics dashboards (mention, don't open every chart)
- Messaging (one line — "in-app messaging between parents and teachers")

### ❌ DO NOT DEMO LIVE
- **Payment gateway** — not built. Frame as **"Razorpay/UPI on the roadmap, available as add-on"** and move on
- **Large list scrolling** — no pagination yet, may stutter
- **Quiz timer** — client-side only, looks weak under scrutiny
- **Real-time channels** — can leak on long-open tabs
- **Hostel / canteen / transport** — frame as *"Phase 2 modules in development"*

## 5. RECORDING WORKFLOW

```
1. Record in 3 takes per Act, not one long take
2. Re-record any Act where you "um", "uh", or stumble
3. Edit: cut all dead air > 1 second
4. Add background music (10% volume, royalty-free, instrumental)
5. Add captions — 70% of buyers watch muted first
6. Export 1080p H.264, under 100 MB
7. Thumbnail: your face + "Save 40% Admin Time" overlay text
```
